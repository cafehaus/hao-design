# haoui 好运业务UI组件库

### 安装组件库

Using npm:
```
npm install haoui --save
```

    Using a script tag for global use:

```html
<script type="text/javascript" src="haoui.umd.min.js"></script>
<link rel="stylesheet" href="dist/haoui.css">
```
 
### 全局导入
    import HaoUI form 'haoui'
    import 'haoui/dist/haoui.css'
 
    Vue.use(HaoUI)
